What is HTTP?
-Hyper Text Transfer Protocol is governs how clients get data from or send data to a server.

What is a URL
- Short for Uniform Resource Locator is an address for some internet resource.

What is DNS?
- is a system that takes human readable URLs and converts them into IP addresses.

What is a query string?
-allows you to pass key value pairs into the URL. For example: ?val1=key1&val2=key2.

List two HTTP Verbs and their cases:
- GET - get some data from the server.
- POST - send some data to the server.

What is an HTTP request ?
- is a request from a client to a sserver which follows the HTTP protocol.

What is an HTTP response?
- is a respone from a server to a client which follows the HTTP protocol.

What is an HTTP Header?
-provide additional information about the request or the response.
-Examples of Request Headers:Expires,Cache Control,Proxy Authenticate, Proxy- Authorization
-Examples of Response Headers:Content-Type, Keep-Alive, Content Encoding, Content-type.

What happens when you type a URL in a browser?
-resolves the name into an IP address using DNS.
-Makes a request to thaat IP address including headers.
-server sends a response.
-makes a DOM from that HTML and finds any other resources needed.
- makes separate HTTP requests for those resources and receives response from the server for each.
